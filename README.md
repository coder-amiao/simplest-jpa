# simplest-jpa
1. 基于QueryDSL语法灵活强大的QueryWrapper,链式QueryChain 强大自由组合查询器,
2. 像写原生SQL一样基于 Java API 查询数据,优雅极致。 
3. 通用service封装,
4. 多租户，自定义全局租户拦截。
